💿 Шаг 2. Создать загрузочную флешку

Запиши образ Windows на USB-накопитель с помощью Rufus — быстро и надёжно.

🔹 Что понадобится?

  • USB-флешка — объёмом от 8 ГБ (все данные на ней будут удалены!).
  • ISO-образ Windows — скачали на прошлом шаге.
  • Программа Rufus — бесплатная, скачать с официального сайта rufus.ie.
⚠️ Внимание: Флешка будет полностью отформатирована. Сохрани все важные файлы с неё заранее!

🔹 Пошаговая инструкция в Rufus

  • 1. Вставь флешку в USB-порт.
  • 2. Запусти Rufus (администратор не обязателен, но можно).
  • 3. В поле «Устройство» выбери свою флешку.
  • 4. Нажми «ВЫБРАТЬ» и укажи свой ISO-образ Windows.
  • 5. Остальные настройки оставь по умолчанию (MBR или GPT — автоматически).
  • 6. Нажми «СТАРТ» и подтверди форматирование.

Процесс займёт 5–15 минут. После завершения флешка готова!

🔹 Альтернатива — Media Creation Tool

Если ты используешь Windows 10, Microsoft предлагает свой инструмент:

  • Скачай «Media Creation Tool» с сайта Microsoft.
  • Запусти и выбери «Создать установочный носитель для другого ПК».
  • Выбери язык, редакцию и архитектуру (64-бит).
  • Выбери «USB-устройство» и укажи свою флешку.
💡 Совет: Rufus даёт больше гибкости и быстрее, но Media Creation Tool — простой вариант для новичков.
