📥 Шаг 1. Скачать образ Windows

Получи официальный ISO-файл с сайта Microsoft — безопасно и бесплатно.

🔹 Где брать образ?

Скачивай только с официального сайта Microsoft, чтобы избежать вирусов и модифицированных сборок.

  • Windows 11 — перейди на страницу загрузки Microsoft и выбери «Скачать Windows 11».
  • Windows 10 — используй официальный инструмент «Media Creation Tool» или скачай ISO напрямую.
  • Всегда выбирай 64-битную (x64) версию — она подходит для большинства современных ПК.
💡 Совет: Если у тебя медленный интернет, можно скачать образ через торрент-раздачу от Microsoft (проверь хеш-сумму SHA-256).

🔹 Какая версия нужна?

Выбирай Windows 11/10 Pro или Домашняя — зависит от твоих задач.

  • Pro — для работы, BitLocker, гипервизор, удалённый рабочий стол.
  • Домашняя — для игр, учёбы и повседневного использования.
  • Если не знаешь — бери Домашнюю, она подойдёт 90% пользователей.

🔹 Проверка загрузки

После скачивания убедись, что файл не повреждён:

  • Проверь размер — ISO весит около 4–6 ГБ.
  • Если используешь браузер — дождись полной загрузки, не прерывай.
  • Для Windows 10 можно использовать Media Creation Tool — он сам скачает и создаст установочный носитель.
⚠️ Важно: Не скачивай Windows с неизвестных сайтов — это частая причина ошибок при установке.
